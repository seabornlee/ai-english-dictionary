### 3.2 AI API成本详细分析

#### 3.2.1 DeepSeek API定价

| 模型 | 输入价格 | 输出价格 | 平均单次查询成本 |
|------|---------|---------|----------------|
| DeepSeek-V2 | ¥1/百万tokens | ¥2/百万tokens | ~¥0.003 ($0.0004) |
| DeepSeek-V3 | ¥2/百万tokens | ¥4/百万tokens | ~¥0.006 ($0.0008) |

#### 3.2.2 成本优化策略

| 策略 | 实施方法 | 预期节省 |
|------|---------|---------|
| **本地缓存** | 常用单词结果缓存7天 | 30-40% |
| **提示词优化** | 精简系统提示词 | 20-30% |
| **批量处理** | 合并多次查询 | 10-15% |
| **分级模型** | 简单词用轻量模型 | 25-35% |
| **结果压缩** | 精简AI输出格式 | 15-20% |

**优化后成本**：可从$0.0004降至$0.0002/次查询

### 3.3 单位经济模型（Unit Economics）

#### 3.3.1 免费用户单位成本

```
免费用户月均成本：
• API调用：500次 × $0.0002 = $0.10
• 服务器分摊：$50 ÷ 1000用户 = $0.05
• 存储成本：可忽略
───────────────────────────────
单免费用户月成本：$0.15
```

#### 3.3.2 付费用户单位经济

```
付费用户（$4.99/月）单位经济：
收入：$4.99
成本：
  • API调用：2000次 × $0.0002 = $0.40
  • 服务器分摊：$0.05
  • 支付手续费（Stripe）：$4.99 × 3% = $0.15
───────────────────────────────
毛利：$4.99 - $0.60 = $4.39
毛利率：87.9%
```

#### 3.3.3 盈亏平衡点分析

```
月度固定成本：$250
付费用户毛利：$4.39/用户
──────────────────────────
盈亏平衡所需付费用户：$250 ÷ $4.39 ≈ 57人

即：每57个付费用户可覆盖当月全部成本
```

### 5.2 关键财务指标

```

• 毛利率：80%
────────────────────────
LTV = $4.99 × 12 × 80% = $47.90

LTV/CAC比率目标：>3:1
即：LTV $47.90 ÷ CAC $15 = 3.2 ✅
```

```
单付费用户回收期：
• CAC：$15（获客成本）
• 首月毛利：$4.39
────────────────────────
回收期 = $15 ÷ $4.39 = 3.4个月 ✅

行业标准：6-12个月，我们优于行业标准
```
